HTML, CSS & JavaScript

What are they and how do they work together.

What is HTML?

HTML is the structure of you document. Using it you can add things such as headings, paragraphs, images and much more.

What is CSS?

CSS adds styling to a web page. Using CSS you can change the font, add color and rearrange items on the page.

What is JavaScript

JavaScript is a programming language. It is the primary language for programming websites. JavaScript adds 'interactivity' to a web page. For example, when you click on a button and something happens, that is JavaScript making things work.
